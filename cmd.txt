VS & GIT commands

#WORKFLOW 
- Crear repo
> git init

> git status

> git add

> git commit -m "Added index.html"

- Comprimir commits en el hash indicado
> git reset {hash}

- Formatear commits hasta el indicado *ojo*
> git reset --hard {hash}

- Moverse entre commits / Siempre que nos movamos entre commits estaremos en headless mode a no ser que volvamos a la rama en este caso 'master'
> git checkout {hash}
> git checkout {rama}

- Recuperar archivo borrado del hardware
> git restore
